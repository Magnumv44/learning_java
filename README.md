### Домашне завдання

###### Завдання №1

> Напишіть програму, яка отримує символи, введені з клавіатури, доки не зустрінеться крапка. Передбачте у програмі лічильник пробілів. Відомості про кількість пробілів повинні виводитися в кінці програми.


###### Завдання №2

> Напишіть цикл **for**, у якому перебиралися значення від **1000** до **0**, з кроком -2.

###### Завдання №3

> Ітераційний вираз для циклу **for** не обов'язково має змінювати значення змінної циклу на фіксовану величину. Ця змінна може набувати довільних значень. Напишіть програму, яка використовує цикл **for** для виведення чисел у геометричній прогресії: **1**, **2**, **4**, **8**, **16**, **32** тощо.

###### Завдання №4

> Код **ASCII** символів нижнього регістру відрізняється від коду відповідних символів верхнього регістра на величину **32**. Отже, для перетворення малої літери в прописну потрібно зменшити її код на **32**. Використовуйте цю особливість для написання програми, яка здійснює отримання символів з клавіатури. При виведенні результатів на екран, дана програма повинна перетворювати **малі літери в великі**, а **великі - в малі**. Інші символи не повинні змінюватися. Робота програми повинна завершуватися після того, як користувач введе **крапку** з клавіатури. І нарешті, зробіть так, щоб програма відображала число символів, для яких було змінено регістр.
